Intelligent Real-Time Phishing
Threat Analysis Platform

PhishGuard is an AI-powered cybersecurity tool that detects phishing URLs in real time using machine learning, domain analysis, and live threat intelligence feeds. Built to protect users from credential theft, identity fraud, and malicious websites.

🔍 Core Features
🤖
Machine Learning Detection
Random Forest classifier trained on phishing datasets. Extracts 20+ URL features and outputs a phishing probability score.
🧬
Brand Similarity (Typosquatting)
Levenshtein distance algorithm detects domains impersonating PayPal, Amazon, Google, Apple, Microsoft, and 11 other major brands.
🌐
Threat Intelligence
Real-time cross-reference against OpenPhish feed and an offline known-phishing domain list. Confirmed phishing URLs are flagged immediately.
📊
Entropy & Heuristic Analysis
Measures domain randomness, checks for IP-based URLs, suspicious keywords, redirect patterns, excessive subdomains, and missing HTTPS.
📷
QR Code Scanner
Upload a QR code image to extract the embedded URL and run it through the full phishing analysis pipeline.
📊
Real-Time Dashboard
Live threat log, scan statistics, verdict distribution chart, and admin tools including data export and backend health check.

🚀 How to Run
1

Install dependencies

2

Train the ML model

python train_model.py — generates phishing_model.pkl

3

Start the Flask API

python app.py — runs on http://127.0.0.1:5000

4

Open the frontend

Open frontend/index.html in your browser — no build step needed.
